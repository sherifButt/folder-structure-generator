# ChatGPT Memory Extension Web App

This project aims to simulate human memory processes to provide long-term memory support for ChatGPT, particularly when dealing with complex tasks such as coding a large application.

## Description

The ChatGPT Memory Extension Web App is built using Nextjs, NextAuth, Stripe, Redux RTK Query, Tailwind, PostgreSQL, Express, and Swagger. It features Input Processing, Title and Description Extraction, Hierarchical Feature Breakdown, Recursive Feature Decomposition, Task List Generation, Mind Map Storage, Context-aware Query Mechanism, Retrieval and Updating Mechanism, Automatic Pruning and Forgetting, and Integration with ChatGPT.

## Parameters

- Input: User input in natural language
- Output: Relevant information retrieved from the memory extension

## Usage Instructions

1. Clone the repository
2. Install dependencies using `npm install`
3. Set up environment variables for authentication and database connection
4. Run the frontend using `npm run dev`
5. Run the backend using `npm run server`
6. Access the web app at `http://localhost:3000`

## Examples

- User input: "What is the purpose of this project?"
- Output: "This project aims to simulate human memory processes to provide long-term memory support for ChatGPT, particularly when dealing with complex tasks such as coding a large application."

## Target Audience

- Developers working with ChatGPT
- Researchers in the field of AI and natural language processing
